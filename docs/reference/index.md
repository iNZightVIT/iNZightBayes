# Package index

## Bayesian Estimation

Functions for Bayesian estimation.

### Conjugate Priors

Functions for conjugate priors.

- [`inz_dNIG()`](https://bayes.inzight.nz/reference/inz_dNIG.md) :
  Normal-Inverse-Gamma Distribution for Prior and Posterior
- [`inz_dbeta()`](https://bayes.inzight.nz/reference/inz_dbeta.md) :
  Beta Distribution for Prior and Posterior
- [`inz_ddir()`](https://bayes.inzight.nz/reference/inz_ddir.md) :
  Dirichlet Distribution for Prior and Posterior
- [`inz_lbinom()`](https://bayes.inzight.nz/reference/inz_lbinom.md) :
  Construct a Likelihood Object for Binomial Data
- [`inz_lmulti()`](https://bayes.inzight.nz/reference/inz_lmulti.md) :
  Construct a Likelihood Object for Multinomial Data
- [`inz_lnorm()`](https://bayes.inzight.nz/reference/inz_lnorm.md) :
  Construct a Likelihood Object for Normal Data
- [`calculate_posterior()`](https://bayes.inzight.nz/reference/calculate_posterior.md)
  : Calculate the Posterior Distribution
